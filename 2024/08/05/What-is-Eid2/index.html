What is E-ID2.0

Author: Jask Don Date: August 4, 2024  16:02:00 Category: Essay

What is E-ID?

E-ID is similar to E-CNY. It is an electronic ID card issued by the state, except that the application scenarios are now very limited. And there are very few platforms that E-ID can be tied to these days. Alipay, for example, Users can bind E-ID on top of Alipay. Nonetheless, it is not accepted by mainstream apps, like WeChat, Jitterbug are not willing to accept it, so much so that it is in a very awkward situation. However, it is undeniable that it does achieve a certain degree of convenience. For example, when I stayed at a hotel, there were still some hotels that recognized E-ID.

So we can learn that E-ID has both benefits and drawbacks.

  • Advantages: convenient and fast

  • disadvantages: Transmission in clear text, not secure. Limited usage scenarios

In general, E-ID is a new product in the electronic information age, but still faces many problems.

National Network Identity（NID)

NID may become a replacement for E-ID, i.e. E-ID 2.0. It uses a new method to prevent leakage of information about a user’s E-ID (though I don’t think that’s feasible). NID generates a kind of netID for registered users that looks like a blockchain public address. Simply put, the ID number is converted into a string of virtual IDs(NET. Nub.)

At this stage, however, NID is only in gray-scale testing and not in public beta, the apps it supports are still very limited.

A number of people have also appeared on the Internet to boycott it because, although it adopts the means of voluntary registration, we cannot rule out the possibility that it will force everyone to register in the future. However, it does defend against the breeding of some false information, such as fraudulent information, rumor mongering, and so on. Unruly elements will gradually diminish because of the NID bind.

Overall, the advantages of NIDs are that they provide better protection against the proliferation of fraudulent information and also protect the privacy of citizens. For example, user A can check in the NID to see which applications his ID card is registered for. However, the disadvantage is also very obvious, if the hacker hacks into the background of NID, it will surely get hold of all the citizens’ information. In short, the network is a double-edged sword, NID can never replace the physical ID card, it can only be used as an auxiliary tool to help people solve some urgent problems, but not as a legal document.
